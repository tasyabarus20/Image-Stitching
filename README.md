## INSTALL PACKAGES

1. Install Imutils
   To install the packages use the command
   ```bash
   pip3 install imutils
   ```
2. Install opencv
   To install it use the command
   ```bash
   Pip3 install opencv-python
   ```
3. Install matplotlib
   ```bash
   Pip3 install mathplotlib
   ```
